Research Project (Group)

Intelligent Daycare Enhancing Child Safety with IoT and ML
ML Part https://github.com/PasinduShyaminda/kidz-care-ml-child-missing.git
Mobile App https://github.com/PasinduShyaminda/kidz-care-mobile-app.git
Website https://pasanchamikara99.github.io/kidzcarewebsite
Demo Video https://www.youtube.com/watch?v=2dlMhYIag9g

KidzCare is an intelligent device built using Raspberry Pi 4 model B integrated with a Pi camera and microphone. It leverages IoT and ML models (YOLOv8) to monitor children, analyze behaviors, and send push notifications with alert history to caregivers via a React Native mobile application. Features include:

  • Child Missing Detection (My Part)
    Check real-time in-class children count with attendance and alert when missing.
  • Fall Detection
    Identifies potential falls and sends immediate notifications.
  • Cry Detection
    Detects when a child is crying and alerts caregivers.
  • Emotion Analysis
    Recognizes and assesses child’s emotions, helping caregivers gauge the child’s well-being.

CookOO is a recipe management web application that allows users to self-register and log in to the application.
